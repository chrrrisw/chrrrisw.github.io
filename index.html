NLA Trove stuff

trove-js

trove-js is a JavaScript library for accessing the National Library of Australia's Trove API. The project website is here and the project code is here.

cite-trove-ff

cite-trove is a Firefox extension to enable a background menu item on the Trove newspaper site. Activating this menu item will scrape some information about the newspaper title, issue and page and copy this to the clipboard along with the selected OCR text. The format and content of the copied text is configurable. The project code is here and the extension is hosted here.
